# ProjectOne

## Project Overview

A Spring Boot REST API for user and post management. Users can create accounts, update their bio, and create posts. Posts are validated against registered users via email lookup before creation.

### Features
- User registration with unique email
- Create, read, and soft-delete posts
- Bio update endpoint
- User existence verification before posting
- Swagger/OpenAPI documentation
- DTO layer for request/response separation

### Tech Stack
- Java, Spring Boot, Spring Data JPA, H2/MySQL, Swagger/OpenAPI, Lombok, Jakarta Validation
